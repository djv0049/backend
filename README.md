# Backend
## Features

- **Ts.ED Framework**: Modern TypeScript framework for Node.js
- **REST API**: User management endpoints
- **TypeScript**: Full type safety
- **Express**: Fast web framework

## Installation

```bash
npm install
```

## Usage

### Development

```bash
npm run dev
```

### Production

```bash
npm run build
npm start
```

- **Ts.ED**: 8.24.1
- **Express**: 5.2.1
- **TypeScript**: 5.9.3
